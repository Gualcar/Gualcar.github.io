Habilidades

Utilização de diversos tipos de linguagens de programação e estratégias lógicas para solução de problemas. A utilização da programação dá liberdade para criar soluções complexas envolvendo diversos softwares, por exemplo, é possível utilizar o Python para integrar o PI System, PIPESIM e excel num único programa.

A solução PI System engloba diversos subsistemas como PI Vision, PI Datalink, PI System Explorer, entre outros. Com o conhecimento de todos esses sistemas é possível criar análises de dados integradas com soluções qualitativas através de gráficos e interfaces e soluções quantitativas através de tabelas e funções estatísticas.

Software de simulação de escoamento multifásico em regime permanente. Através da utilização do PIPESIM é possível entender o comportamento dos fluidos através da geração de perfis de pressão e temperatura, além disso, é possível modelar diversos tipos de poços de petróleo e layouts de produção.

Simulador de escoamento multifásico transiente. Um dos softwares mais robustos para modelagem e simulação do escoamento, apresenta diversas soluções para problemas operacionais como passagem de PIG, pressurização/despressurização de linhas, escoamento em slugs, partida e parada de poços, dentre outras.

Software para modelagem de operações unitárias. Através do HYSYS é possível modelar e simular diversas operações unitárias amplamente utilizadas na indústria química/petroquímica como cálculo de separação em vaso flash, separação em torrer de destilação e de contato, aquecimento/resfriamento de fluidos. Além disso, é possível realizar diversas análises do fluido através de sua composição.
